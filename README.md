# DentDigital

A full-service digital marketing agency website built with **Next.js 15**, **TypeScript**, and **Tailwind CSS** — ready for deployment on **Vercel**.

## Pages

| Route | Description |
|---|---|
| `/` | Home – hero, stats, services overview, portfolio preview, testimonials, CTA |
| `/about` | About – story, mission, values, team |
| `/services` | Services – full breakdown of all offerings + process |
| `/portfolio` | Portfolio – detailed case studies |
| `/contact` | Contact – enquiry form + contact details |

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/beckhie1/DentDigital)

The simplest way to deploy is via the [Vercel Platform](https://vercel.com). The repo includes a `vercel.json` configuration file.
